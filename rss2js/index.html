RSS Feeds via JavaScript

Criado por Alan Levine, Setembro 2003
Traduzido toscamente por Charles Pilger, Novembro 2003

Implementar RSS feeds em seu próprio site é simples se o código é construído dentro de um sistema como o Radio ou o MovableType, mas quem é capaz de sentar e sair escrevendo código para interpretar XML? Eu não.

Eis aqui uma maneira de você facilmente inserir dinamicamente RSS atualizados em qualquer página web, blog, ou CMS (Course Management System).

Ele faz uso de um script PHP (versão demo rodando no nosso servidor) que interpreta o feed XML e retorna um conjunto de comandos de escrita Javascript que inserem a informação na sua página. Tudo que você precisa fazer é inserir um simples linha de código Javascript na parte da sua página onde você quer o feed. O única outra coisa a fazer é linkar ou inserir um style sheet para formatar a saída.

Exemplo

Aqui está a exemplo com os 10 itens mais atuais no feed da Maricopa Learning eXchange:

O código para colocar este na sua página é:

<script language="JavaScript" src="http://jade.mcli.dist.maricopa.edu/feed/rss2js.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml">
</script>
<!-- alternative for no javascript -->
<noscript>
<a href="http://jade.mcli.dist.maricopa.edu/feed/rss2html.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml">View RSS feed
</noscript>

onde o valor de "src=" é a URL para o RSS feed. Repare que providenciamos uma opção NOSCRIPT para usuários com JavaScript disabilitado ou para aqueles que usam leitores de tela.

Opções Adicionais

Nós adicionamos mais alguns parâmetros para controlar a saída do seu feed.

Mostrar/esconder a informação do canal (chan=yes|no|title)
O padrão é mostrar o título do canal e a sua descrição (chan=yes). Você pode ainda especificar para mostrar apenas o título do canal (chan=title). Para esconder esse dado ou para ajustá-la somente ao título, adicione o parâmetro a sua URL:

<script language="JavaScript" src="http://jade.mcli.dist.maricopa.edu/feed/rss2js.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&chan=no">
</script>
<!-- alternative for no javascript -->
<noscript>
<a href="http://jade.mcli.dist.maricopa.edu/feed/rss2html.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&chan=no">View RSS feed</a>
</noscript>

Limitar p número de ítem retornados (num=X)
O padrão é mostrar todos os itens do feed. Para retornar apenas um determinado número de itens, especifique-o como:

<script language="JavaScript" src="http://jade.mcli.dist.maricopa.edu/feed/rss2js.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&num=3">
</script>
<!-- alternative for no javascript -->
<noscript>
<a href="http://jade.mcli.dist.maricopa.edu/feed/rss2html.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&num=3">View RSS feed</a>
</noscript>

O valor padrão ou a especifocado como num=0 mostrará todos os items.

Mostrar/esconderdescrição dos itens (desc=yes|no)
O padrão é esconder as descrições dos artigos. Para visualizar a descrição do item, adicione o parâmetro a sua URL:

<script language="JavaScript" src="http://jade.mcli.dist.maricopa.edu/feed/rss2js.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&desc=yes">
</script>
<!-- alternative for no javascript -->
<noscript>
<a href="http://jade.mcli.dist.maricopa.edu/feed/rss2html.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&desc=yes">View RSS feed</a>
</noscript>

Mostrar/esconder data de postagem do item (date=yes|no)
O padrão é esconder a data dos artigos. Para mostrar a data de publicação do item adicione o parâmetro a sua URL:

<script language="JavaScript" src="http://jade.mcli.dist.maricopa.edu/feed/rss2js.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&date=yes">
</script>
<!-- alternative for no javascript -->
<noscript>
<a href="http://jade.mcli.dist.maricopa.edu/feed/rss2html.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&date=yes">View RSS feed</a>
</noscript>

Construindo um Feed!

Não se preocupe em escrever o código! Este formulário irá ajudá-lo a construir um feed. Tudo que você precisa para o código RSS é a URL.

URL? [ Entre com o endereço da sua feed RSS ]

Mostrar canal? [ Mostra informação sobre o fonte do feed ]
yes no

Numero de itens? [ Entre com o número de itens que serão mostrados (entre com 0 para mostrar todos) ]

Mostrar descrição dos itens? [ Descrição dos itens (não significa que apenas o título é mostrado) ]
sim não

Mostrar data de postagem do item? [ mostra quando o item foi adicionado ao feed ]
sim não

Veja tudo isso em ação...
Aqui vai um exemplo de uso de todos os parâmetros, mostrando apenas o título do canal (chan=title), limitando o número de itens a 3 (num=3), mostrando as descrições (desc=yes) e mostrando a data de postagem (date=yes):

<script language="JavaScript" src="http://jade.mcli.dist.maricopa.edu/feed/rss2js.php?src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&chan=title&num=3&desc=yes&date=yes"></script>
</script>
<!-- alternative for no javascript -->
<noscript>
<a href="http://jade.mcli.dist.maricopa.edu/feed/rss2html.php?
src=http://www.mcli.dist.maricopa.edu/mlx/feed/new10.xml&chan=title&num=3&desc=yes&date=yes">View RSS feed</a>
</noscript>

Deixando bonito

Insira um link para este arquivo de estilo ou coloque o conteúdo diretamente no seu arquivo HTML, entre as tags <style> ... </style> . Edite para fazer o design do seus sonhos (precisa suar um pouco no CSS, mas não muito com as cores e as fontes).
